Keywords

“Watchy” Angular SPA

  • angular
  • jest
  • typescript
  • indexedDB

Watchy is an app that allows users to create lists of movies and TV shows. It uses TMDB's API to get movie and TV show details and saves the lists locally to indexedDB using Dexie.js so there's no need for user accounts or a back-end server.

Although the app is written in Angular I configured it run jest unit tests instead of jasmine as a personal preference. The app has 100% unit test coverage and I'm currently working on end-to-end tests using Cypress.

Accessible Custom Elements (ACE) library

  • accessibility
  • typescript
  • cypress
  • node.js

ACE is an open-source library of UI components accessible to users who rely on assistive technologies or keyboard navigation. The components conform to W3's WAI-ARIA Authoring Practices. The goal of ACE is to help developers create beautiful and fully inclusive websites.

I led the project from concept to publication on npm and use in Potato’s website and various Google web apps, and I'm the project maintainer.

The components were created using TypeScript and tested extensively using Cypress. I also wrote detailed documentation and use Node.js, Gulp and CI/CD to automatically update the documentation webpages with the latest code and interactive demos of each component.
